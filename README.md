# 🌸 Daily Quote Generator

A simple Flask-based web application that generates random motivational quotes and displays them on randomly selected background images.

## ✨ Features

- 🎲 Random quote generation
- 🖼️ Random background image generation
- 🔄 Regenerate button for a new quote
- 🏠 Home button for easy navigation
- 🎨 Custom-designed user interface
- ⚡ Built using Flask routing and Jinja templates

## 🛠️ Tech Stack

- Python
- Flask
- HTML5
- CSS3
- Git & GitHub

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Apurvimaurya/Daily-quote-generator.git
```

2. Navigate to the project directory

```bash
cd Daily-quote-generator
```

3. Install Flask

```bash
pip install flask
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
